# action-agent-document
action para integrar agent que realiza documentacion de los cambios realizado en repositorio.
 
## Example usage

```yaml
on: [push]

jobs:
  aagent:
    runs-on: ubuntu-latest
    steps:
      - name: agent
        uses: pocagentai/action-agent-document@v1.0
```
